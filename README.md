# P_Wingtto
DX11 자체 엔진을 이용한 윙또 3D 게임

## 깃 커밋 규칙 (Git Commit Comment)
```
type: Title

body(Optional)
```
```
Types
+ feat : 기능 추가
+ fix : 오류 수정
+ docs : 문서 추가
+ assset : 에셋 추가
```

## 프로젝트 폴더 규칙 (Project Folder Rules)
```
Libraries : 외부 종속 파일들
└ Lib : 동적 및 정적 라이브러리
└ Include : 헤더 포함 파일

Engine : 엔진 프로젝트
└ Core : 핵심 엔진 코드
└ Refelection : C++ 리플렉션 코드

ConsoleTest : 콘솔 테스트 프로젝트

Client : 클라이언트 프로젝트

Resources : 에셋
```
